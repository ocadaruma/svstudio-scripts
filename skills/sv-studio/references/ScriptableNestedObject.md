# ScriptableNestedObject

## ScriptableNestedObject

`ScriptableNestedObject` is an extension of [`NestedObject`](NestedObject.md) that allows the storage of user-defined attributes inside the object. It is useful for storing internal data used by the script persistently.

Any data stored into `ScriptableNestedObject` will be copied and pasted along with the object itself, and will be retained after saving and re-loading the project.

(supported since 2.1.2)
