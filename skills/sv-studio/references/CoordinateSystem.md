# CoordinateSystem

## CoordinateSystem

A UI state object for navigating a two-dimensional scrollable area where the x-axis is time and the y-axis is value.

It is used in piano roll ([`MainEditorView`](MainEditorView.md)) and arrangement area ([`ArrangementView`](ArrangementView.md)). In both cases, the unit for the x-axis is blicks. However, the arrangement area only uses `CoordinateSystem` for the x-axis.
